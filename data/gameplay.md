## 1. 主题曲

> **关键信息**：是否消耗理智=是 | 玩法类型=主线剧情 | 最新章节=第十五章起合并为曲谱

### 简介

主题曲是游戏主线故事的关卡，承载游戏核心剧情。玩家通过通关主题曲章节推进游戏进度，解锁更多功能。

### 重要说明

- 自主题曲第十五章实装起，原【主题曲】【插曲】【别传】及【故事集】合并为**曲谱**
- 取消按活动形式的分类，改为按故事线分类的【乐章】
- 不再区分"主题曲章节"与"活动"，统一表述为【章节】
- 若干个直接相关的【章节】将一并收录至对应的【乐章】，乐章之间会以导入/导出分支的方式互相关联

### 分类说明

| 分类 | 说明 |
|------|------|
| 主题曲 | 对应曲谱乐章【Main Story/主题曲 为了明日】 |
| 插曲 | 记录与主题曲乐章直接相连的曲谱乐章及其章节 |
| 别传 | 记录与主题曲乐章不直接相连的曲谱乐章及其章节 |

### 游玩入口

玩家可以通过曲谱右上角的【乐章收录】按钮进入章节总览页面。

## 2. 活动关卡

> **关键信息**：是否消耗理智=是 | 分类=插曲/别传/故事集 | 获取奖励=蚀刻章(限时) | 解锁消耗=事相结晶

### 曲谱分类

自主题曲第十五章实装起，活动关卡分为以下几类：

#### 插曲

- 和主线有紧密联系的SideStory活动关卡
- 活动首次开放结束后立刻收录至曲谱
- 复刻活动期间，曲谱中的活动入口变更为复刻活动入口，活动结束后还原
- 删除了活动配套的商店、任务、限定奖励和代币掉落
- 部分材料掉落相较于原活动有更改
- **非活动期间不可获得蚀刻章，活动复刻结束后永久无法获得蚀刻章**
- 如未曾体验过某个插曲的关卡，需要消耗1个事相结晶以解锁该插曲
- 对于开放了"记录修复"功能的插曲，完成关卡（不含剧情、教程关卡）积累评价星数可获取限定奖励

#### 别传

- 其它的SideStory活动关卡
- 仅在复刻结束后收录至曲谱
- 删除了活动配套的商店、任务、限定奖励和代币掉落
- 部分材料掉落和理智消耗相较于原活动有更改
- **不可通过相关关卡获得蚀刻章**
- 如未曾体验过某个别传的关卡，需要消耗1个事相结晶以解锁该别传
- 对于开放了"记录修复"功能的别传，完成关卡积累评价星数可获取限定奖励

#### 故事集

- 限时故事集活动，如「十字路口」等
- 具体活动内容和奖励请参考当期活动公告

### 特别活动：卫戍协议：盟约

当前限时活动示例：

- 模拟数据关卡：AC-TR-1入门协议、AC-1至AC-4标准/险境/绝境/终极模拟
- 地图数据和敌方势力随机搭配
- 提供多种战场选择

## 5. 保全派驻

> **关键信息**：访问方式=终端-常态事务 | 是否消耗理智=否 | 主要获取=模组升级材料 | 解锁条件=模组+1-5通关

### 简介

保全派驻模式是以类集换式卡牌游戏的玩法为主的游戏模式，**常驻开放**。

### 访问方式

终端 - 常态事务

### 解锁条件

1. 开启至少一件干员的专属模组
2. 通关主题曲1-5
3. 完成此模式的保全演练关卡即可正式参与保全派驻作战

### 特色

- 通过此模式可以获取模组升级所使用的材料：**数据增补条**和**数据增补仪**
- 以上材料也可通过其它途径获取
- **本模式不消耗理智**

### 危险等级效果

每1级危险等级使敌人的生命提升25%、攻击力提升5%、防御力提升5%。

### 定向导能元件

可用的定向导能元件包括：

| 元件 | 被动效果 | 主动效果 |
|------|----------|----------|
| 能量扰动放大器 | 敌人处于停顿、束缚、眩晕、寒冷、冰冻、浮空异常状态时，防御力和法术抗性逐渐降低（最多降低50%） | 放置设备，周期性对范围内所有敌人随机施加异常效果 |
| 竞技记录分析仪 | 卡西米尔干员撤退时直接返回再部署区 | 减少再部署区所有卡西米尔干员5点部署费用 |

## 7. 生息演算

> **关键信息**：访问方式=终端 | 是否消耗理智=否 | 玩法类型=回合制经营 | 最新主题=沙洲遗闻

### 简介

生息演算是强调决策性的回合制经营玩法，分主题且一段时间内每月更新的探索型常驻活动。

### 最新主题：沙洲遗闻（已完结）

> 舒巴特-阿尔萨兰，一片位于萨尔贡的失落之地。黄沙漫过堤坝，堵塞了往昔的河道，家园逐渐荒废。原以为仅剩废墟的这里，家园与文明的篝火却未曾熄灭。沉默的沙洲，坚守的人们，神秘的过往，这一切正等待着我们的到来。

- 常驻奖励：繁荣馈赠
- 限时奖励：战争浪潮（繁荣点数）

### 特有概念

#### 演算资源分类

| 类型 | 示例 |
|------|------|
| 食材 | 蟹腿、圆圆肉片等（可烹饪） |
| 素材 | 矿石、木材、水等（用于建造和研究） |
| 货币 | 粗制赤金等（用于交换物资） |

#### 时间系统

| 概念 | 说明 |
|------|------|
| 生息日 | 玩家的回合数，每日光消耗一定次数的决断 |
| 生息周 | 每3个生息日为一个生息周 |
| 量定日 | 生息周的最后一日，结算本周期得分 |

#### 决断

- 每一生息日玩家拥有一定的决断次数
- 决断次数初始为每天2次
- 消耗决断次数可以执行动作
- 进入新一天后决断次数重新补充至上限

### 地图结构

| 区块类型 | 说明 |
|----------|------|
| 驻扎地 | 基地所在处，可进行自由建设，必须守卫的核心 |
| 哨站 | 驻扎地外围的前线缓冲地带 |
| 旷野房车 | 特殊行动中的临时前进营地 |
| 后舍 | 友人来访的地方，不消耗决断 |
| 冲突区 | 需要战斗的区域 |
| 资源区 | 获取演算资源的关卡，有固定数量上限 |
| 捕猎区 | 获取食材的关卡 |
| 奇遇 | 遇到不同的事件 |
| 集结区 | 派遣干员参与额外任务 |
| 荒废城镇 | 驻守敌人和特殊装置，需要击破号令点 |
| 要塞 | 驻守精英敌人，击败后获得丰厚奖励 |

| 关隘 | 需要击破机关石门才能继续前进 |

### 特殊作战机制

#### 迷雾

- 整个地图通常被迷雾覆盖
- 敌人持有不可选中，仅可见位置和移动行为
- 通过我方单位提供视野来点亮地块
- 曾被点亮的地块将永久记录

#### 敌袭系统

| 类型 | 说明 |
|------|------|
| 走卒敌袭 | 固定种类与数量的普通敌人 |
| 精英敌袭 | 固定种类与数量的精英敌人 |
| 领袖敌袭 | 包含领袖单位的敌袭 |
| 游勇敌袭 | 初始只有约40%敌人，逐渐发展至完整状态 |
| 流寇敌袭 | 向已被玩家击破的荒废城镇移动 |

### 行为系统

1. **数据存取**：支持存档/读档功能，最多保留2个快照
2. **干员出击**：同一干员同一生息日仅能出击一次
3. **后勤入驻**：设置进驻干员为下个生息周提供战斗增益
4. **食品制作**：烹饪食物临时增强干员战斗能力
5. **资源采猎**：进入资源区/捕猎区获取材料
6. **建筑设施**：建造和升级设施

### 特色

- **游玩本模式不消耗理智**
- **不计入模组任务进度**
- 存在关卡内单位数量众多导致设备掉帧的情况

### 核心机制

#### 指标系统

玩家需要从路径图上选择指标来增加难度：

- 虚线路径表示当前状态下无法连接的路径
- 虚线指标表示当前状态下无法选中的指标
- 可以在一次作战中建立多个通路以选择更多指标
- 取消选择时，与之相连的指标也会被一同取消

#### 指标评分

- 各个指标有各自的指标评分
- 参与当次作战的指标评分之和即为当次作战的指标总分
- 指标评分会细化到六个不同的维度，以六维图方式显示

#### 指标集

- 具有强关联性的指标群被划分到一个指标集内
- 当玩家在指标集内选择的指标达到该指标集的最高分数时，即视为选择了该指标集
- 部分指标集可以提供额外分数

| 阵眼任务 | 需要在通关时满足给定的任务条件，完成后可领取奖励 |

## 9. 矢量突破

> **关键信息**：访问方式=终端-周期挑战 | 是否消耗理智=否 | 玩法类型=驻防挑战 | 当前批次=#2巫术之夜

### 简介

矢量突破结合了联锁竞赛和纷争演绎的特点，玩家将带领干员通过训练官设置的层层考验。

### 最新批次

- **#2 巫术之夜**（最新）

### 核心区域

#### 核心突破

- 划分为难度逐级增加的**8-12个层级**
- 玩家将从第1层开始逐级挑战
- 每次通关后，下一层都会追加新的考核点
- 所有关卡的关卡生命值为**1**，必须完美完成指挥作战方可通过
- 每突破一个层级将获得大量当期积分点
- 重复完成当前层级行动也可以获得少量积分点

#### 全力以赴

- 在完成全部核心突破关卡后，在活动的下半阶段可以挑战的关卡
- 有机会遭遇往期的训练官

#### 特别战线

通过核心突破第1层级解锁。

**功能**：

- 当在核心突破作战中面临难以攻破的局面时，可以前往特别战线激活战备补给
- 存在普通驻防点与高级驻防点
- 必须先占据普通驻防点才能挑战高级驻防点
- 玩家无法使用助战干员

**驻防规则**：

- 每个控制点对战有时间限制，必须在时限内处理所有驻防单位
- 夺取控制权后，干员将驻防在当前控制点
- 驻防的干员将无法参与核心突破作战
- 可以随时选择撤离或替换驻防干员

**补给类型**：

- 不同控制点可派出迎战的干员数量上限不同
- 高级驻防点可提供更高级的战备补给
- 激活补给存在上限，按装配表从下到上的顺序生效

### 成果展示

可以浏览以下项目：

- 当赛季的核心突破成果，包括最高突破层数、补给携带状况、编队干员等
- 当赛季的全力以赴突破成果
- 当赛季的蚀刻章获取情况
- 当赛季中特别战线的控制夺取记录

### 特色

- 战斗允许在暂停时选择以当前阵容重新开始
- 自#2起，可在驻防总览页面分享驻防
